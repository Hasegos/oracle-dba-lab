## 🎯 프로젝트 목표

1. 기본 SQL 문법 숙달: SELECT, JOIN, GROUP BY 등 자주 쓰는 SQL 구문을 문제 풀이를 통해 반복 연습합니다.
2. Oracle 환경 기준 SQL 정리: ROWNUM, TO_CHAR 등 Oracle에서 자주 사용하는 문법을 실제 문제에 적용하며 익힙니다.
3. 문제 풀이 기록 습관화: 테스트용 DDL/DML과 쿼리 작성 과정을 함께 정리하여, 나중에 다시 보며 참고할 수 있도록 기록합니다.
