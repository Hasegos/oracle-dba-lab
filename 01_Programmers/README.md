## 🎯 프로젝트 목표 (Goals)

1. **DB 쿼리 역량 강화:** SELECT, JOIN, GROUP BY 등 필수 SQL 구문을 숙달하고 다양한 문제 유형에 적용합니다.
2. **Oracle 환경 이해:** `ROWNUM`, `TO_CHAR` 등 특정 데이터베이스(Oracle) 환경에서 요구하는 특수 함수 및 문법 처리 방식을 익힙니다.
3. **체계적인 문서화:** 단순한 답안 제출을 넘어, **테스트 환경 구축** 및 **문제 풀이 과정 주석**을 통해 검증 가능하고 가독성 높은 코드를 작성하는 습관을 정립합니다.
