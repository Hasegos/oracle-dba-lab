# 📌 01. SELECT 모듈 학습 관리 가이드

> 이 디렉토리는 Programmers SQL Kit의 SELECT 모듈 문제를 Oracle 기반으로 학습한 공간입니다.<br>
> 단순히 문법 암기가 아닌 **문제를 통해 '조건 설정 → 정렬 → 함수 → NULL 처리' 패턴을 자연스럽게 익히는 것**을 목표로 합니다.

---

## 🎯 학습 범위 (Focus Scope)

이 모듈에서 중점적으로 다룬 기술 및 구문은 다음과 같습니다.

* 단일 테이블 조회 및 `DISTINCT` 구문 이해
* `WHERE` 절을 활용한 복합 조건 및 범위 필터링 (`AND`, `OR`, `IN`, `LIKE`)
* `ORDER BY`를 활용한 다중 기준 정렬
* NULL 값 처리 (`IS NULL`, `IS NOT NULL`, `NVL`)
* 문자열/숫자/날짜 관련 기본 함수 활용 (`ROUND`, `TRUNC` 등)
* **Oracle 특화:** `ROWNUM`을 서브쿼리와 함께 활용한 상위 N개 레코드 추출 방법론

---

## 📁 디렉토리 구조 및 목표 (Level Breakdown)

| 폴더 | 목표 난이도 | 포함 내용 및 학습 초점 |
| :---: | :---: | :--- |
| **Level1** | 기본/쉬움 | SELECT 기본 문법, 단일 조건 및 단순 정렬 패턴 익히기 |
| **Level2** | 보통 | 다중 조건 필터링, 기본적인 함수 활용 및 NULL 값 처리 적용 |
| **Level4** | 심화 | `ROWNUM` 활용, 고난도 조건 및 함수 조합 등 심화 SELECT 패턴 구현 |

---

## 📝 문제 풀이 기록 방식 (SQL File Format)

이 폴더 내의 모든 `.sql` 파일은 `테스트 환경 구축부터 답안 도출` 까지 포함하는 일관된 형식을 유지합니다.<br>
파일 상단에는 아래 4가지 필수 기록 요소를 주석 형태로 포함해야 합니다.

### 4가지 필수 기록 요소

| No. | 기록 요소 | 설명 및 포함 내용 |
| :---: | :--- | :--- |
| **1** | **문제 명 및 난이도** | 문제 제목과 출처(Programmers/Level) 명시 |
| **2** | **핵심 요구사항 요약** | 문제 해결을 위해 최종적으로 도출해야 할 결과 정의 |
| **3** | **주요 기술 및 함수** | 이 문제 해결에 사용된 핵심 SQL 구문/함수 |
| **4** | **구현 논리 및 단계** | 문제 분석부터 답안 도출까지의 사고 과정 (예: 필터링 → 집계 → 포맷팅 순서) |
