# 📌 01. SELECT 문제 풀이 정리

> ⬅️ **[Programmers 메인으로 돌아가기](../README.md)**
>
> 이 디렉토리는 Programmers SQL Kit의 SELECT 유형 문제를 **Oracle 기준으로 직접 풀이·정리한 기록**입니다.  

---

## 🎯 학습 범위

이 부분에서는 **SELECT 기본기를 실제 쿼리 작성 기준으로 정리**합니다.

- 단일 테이블 조회 및 `DISTINCT`
- `WHERE` 절을 이용한 조건 처리 (`AND`, `OR`, `IN`, `LIKE`)
- `ORDER BY` 다중 정렬
- NULL 처리 (`IS NULL`, `IS NOT NULL`, `NVL`)
- 숫자/문자/날짜 기본 함수 (`ROUND`, `TRUNC` 등)
- **Oracle 기준 상위 N개 조회: `ROWNUM + 서브쿼리` 패턴**

---

## 📝 필수 기록 항목

모든 `.sql` 파일은 **“테스트 데이터 생성 → 쿼리 작성 → 결과 검증”까지 한 파일에서 확인 가능하도록 관리합니다.**

| No | 항목 | 내용 |
