# 📚 Oracle SQL Practice for DBA

이 저장소는 **Oracle 환경 기반 SQL 학습 내용을 정리한 개인 포트폴리오**입니다.<br>
프로그래머스 SQL Kit 문제 풀이와 Oracle 특화 SQL 패턴 정리를 통해  
**Oracle 환경에 익숙한 SQL 작성 능력을 단계적으로 정리하고 있습니다.**

- 대상 DB: Oracle  
- 학습 범위: DQL, 집계 함수, 서브쿼리, JOIN, 계층형 쿼리  
- 목적: **운영 DBA / 개발 DBA 준비를 위한 SQL 기초 역량 정리**

---

## ✔️ 학습 및 기록 기준

이 저장소의 모든 SQL 문제 풀이는 다음 기준을 따릅니다.

- ✅ **DDL + DML + 정답 쿼리 전체 포함**
- ✅ **실제 데이터 기준으로 실행 가능한 형태 유지**
- ✅ **문제 해결 흐름을 주석으로 간단히 정리**

## 🧾 SQL 파일 표준

모든 SQL 파일 상단에는 아래 4가지 항목이 반드시 기록됩니다.

1. 문제 정보 및 출처  
2. 핵심 요구 조건 요약  
3. 사용한 핵심 함수 및 문법  
4. 쿼리 설계 흐름 및 논리 정리  


→ **학습 과정과 실행 흐름을 다시 따라가 볼 수 있도록 기록하는 것을 기준으로 합니다.**
